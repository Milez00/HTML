Игрушечный динозавр

Фотограмметрия • 2026

Данная модель была создана методом фотограмметрии. Для реконструкции использовалась серия фотографий игрушечного динозавра, после чего модель была обработана и оптимизирована для просмотра в браузере.

История модели

Проект демонстрирует возможности современной фотограмметрии по созданию высокодетализированных 3D-моделей реальных объектов. После обработки модель была очищена, ретопологизирована и экспортирована в формат GLB.

Особенности

  • Создана методом фотограмметрии.
  • Оптимизирована для WebGL.
  • Поддерживает просмотр в дополненной реальности (AR).
  • Формат модели: GLB.

Использованные технологии

  • Фотограмметрия
  • Blender
  • model-viewer
  • HTML / CSS

Интересный факт

Современные методы фотограмметрии позволяют получать миллионы полигонов всего из нескольких десятков или сотен фотографий, а затем автоматически создавать текстуры высокого разрешения.
